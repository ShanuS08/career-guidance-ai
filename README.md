# 🎯 Career Guidance AI System

## 📌 Overview
This project is an AI-based career recommendation system that suggests suitable career paths based on user skills and interests.

## 🚀 Features
- Skill-based career prediction
- Interest-based analysis
- Simple and interactive web UI
- Flask-based lightweight web app

## 🧠 Logic Used
Rule-based classification system using keyword matching for:
- Programming languages
- Technical skills
- Career interests

## 🛠️ Tech Stack
- Python
- Flask
- HTML
